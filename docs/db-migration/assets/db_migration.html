What changes: the later stages stop treating the filesystem as the database. evaluate and categorise write structured rows instead of .md/.json blobs, so cross‑property questions (ADR ranges, archetype membership, freshness, dedupe) become queries instead of directory walks. What stays a file: the raw collect dossiers, binary source PDFs, and the generated *-pricing.py scripts — these are sources or reproducible artifacts, not query targets. Markdown can still be exported from the DB for human review. Collect is untouched.
